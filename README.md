# Trabajo Prácticas con Variables en JavaScript

## Descripción 📝
Este trabajo práctico contiene una serie de ejercicios básicos de **JavaScript** orientados a practicar el uso de **variables**.

Los ejercicios están organizados por niveles de dificultad y ayudan a comprender conceptos fundamentales de programación.

---

## Tecnologías utilizadas 📎
- JavaScript
- HTML (para ejecutar los scripts en el navegador)
